# Esercizio 5
All'interno del file template "le-mie-ricette.php" utilizzare la funzione "get_posts_that_belongs_to_recipes_category()"
per recuperare gli articoli appartenenti alla categoria "Ricette".

Per ogni articolo bisogna stampare le seguenti informazioni:
* Immagine dell'articolo (suggerimento: l'immagine di anteprima su WordPress si chiama thumbnail. Serve l'url per far visualizzare l'immagine.)
* Titolo dell'articolo all'interno di un tag `<h1></h1>`
* Descrizione dell'articolo all'interno di un tag `<p></p>`
* Il titolo dell'articolo deve contenere l'url che rimanda alla pagina dell'articolo.
